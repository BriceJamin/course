## TITRE : Les chaînes de caractères 

## chaînes de catactères 

Nous allons voir dans cette vidéo le type str qui représente
les chaînes de caractères. Le type str est un type séquence, 
c'est-à-dire que toutes les opérations et fonctions sur les 
séquence s'appliquent aux chaînes de catactères. Je vous
rappelle rapidement que les principales sont le test 
d'appartenance [x in s], ou [x not in s], la concaténation 
[s + t], l'accès par indice [s[i]], le slicing [s[i:j:k]] qui
permet d'obtenir la sous chaîne allant de i à j-1 par pas de k. 
Je rappelle également que les chaîne de caractères sont immuable, 
ce qui veut dire qu'une fois qu'une chaîne de caractères est 
créée, on ne peut plus la modifier. 

[[
x in s
x not in s
s + t
s[i]
s[i:j:k]
]]
